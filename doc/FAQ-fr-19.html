19. Mais où sont les rapports prédéfini

Parce que les rapports peuvent toujours être facilement adaptés, ils sont inclus dans le produit mais pas installé. Pour les trouver, allez simplement dans le répertoire contrib/rapport et éxécutez les scripts sql soit grâce à pgAdmin soit en commande en ligne soit avec l'outil PhpPgAdmin.
