13. Comment appliquer un patch ?

13.1 Quel patch utiliser ?

Dans certains cas, le dernier patch ne reprend PAS les autres, il faudra lire la release note du patch pour savoir ce qu'il faut appliquer avant. En règle générale, il faut appliquer les patch dans l'ordre : le premier, puis le second...

Sinon, les patchs sont toujours nommés ainsi path-(numéro_version)-(numéro_de_path).tgz

exemple path-r092-1.tgz, est le premier patch pour la release 0.9.2

13.2 Comment appliquer le patch

Après l'avoir télécharger, allez dans le répertoire de PhpCompta, c'est-à-dire celui qui contient les sources

  1. Décompresser le patch (gzunip patch-...)
  2. puis appliquez le patch -p0 < patch...
